# Magic Item JSON formats
<br>

## Potion

```json
{
    "name": "name",
    "type": "potion",
    "rarity": "rarity",
    "homebrew": false,
    "details": "Item details"
}
```
<br>

## Spell Scroll

```json
{
    "name": "name",
    "type": "scroll",
    "rarity": "rarity",
    "spells": [""],
    "homebrew": false,
    "details": "Item details"
}
```

## Every other type:

```json
{
    "name": "name",
    "type": "type",
    "rarity": "rarity",
    "attunement": false,
    "attunement_type": "None",
    "class": "None",
    "race": "None",
    "other": "None",
    "variations": [""],
    "homebrew": false,
    "details": "Item details"
}
```
